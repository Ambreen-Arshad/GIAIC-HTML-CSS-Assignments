Chapter-3 List Tags

There are four ( 4 ) types of List:

  1. Order List => < ol > < /ol >
  2. Un-order List => < ul > < /ul >
  3. Discription List => < dl > < /dl >
  4. Nested List => < ol > < ul >......< /ul >
    < /ol >
